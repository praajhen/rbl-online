# Rule-Based Learning Online Experiment

This repository contains an online behavioral task designed to study rule-based learning using visual stimuli.

## Task Overview

Participants view shapes in different colors and must learn the correct response category using feedback. The correct rule is not explicitly stated and must be discovered during the task.

The experiment consists of:

* Practice block
* Three main blocks with increasing difficulty

Participants respond using keys 1, 2, or 3.

## Trial Structure

Each trial includes:

* Fixation cross
* Stimulus presentation
* Response window
* Feedback (Correct / Incorrect)

Participants use feedback to learn the correct responses.

## Data Recorded

The following information is recorded:

* Participant ID
* Block
* Trial number
* Stimulus identity
* Response
* Accuracy
* Reaction time
* Score

Data are automatically saved to a connected Google Sheet.
